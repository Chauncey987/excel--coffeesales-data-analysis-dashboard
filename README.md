# excel--coffeesales-data-analysis-dashboard
End-to-end Excel data analysis project demonstrating data cleaning, pivot table analysis, and dashboard design for business decision-making.

## Overview
This project demonstrates an end-to-end data analysis workflow using Microsoft Excel, focusing on transforming raw data into clear, actionable business insights through structured analysis and visualization.

The project includes data cleaning, exploratory analysis, pivot table development, and dashboard creation to support data-driven decision-making.

---

## Business Objective
The objective of this analysis is to:
- Clean and organize raw transactional data
- Identify trends and patterns that impact performance
- Create a dashboard that summarizes key metrics for stakeholders
- Present insights in a clear, business-friendly format

---

## Tools Used
- Microsoft Excel
  - Data cleaning and formatting
  - Calculated fields and formulas
  - Pivot tables
  - Charts and dashboard design

---

## Dataset
The dataset used in this project represents a simulated business environment and includes:
- Transaction-level data
- Product and customer attributes
- Revenue and quantity metrics

All data is used for educational and portfolio purposes only.

---

## Analysis Process
1. Data cleaning and preparation
2. Creation of calculated fields
3. Pivot table analysis
4. Dashboard development
5. Insight extraction

---

## Key Insights
- Performance varies significantly across product categories
- Time-based trends reveal periods of higher and lower activity
- Dashboard visualizations help quickly identify strengths and gaps

---

## Deliverables
- Excel dashboard file
- Cleaned dataset
- Visual preview of dashboard results
